Primer acercamiento al desarrollo Front-End

Objetivo:

El objetivo de esta actividad es entender como funcionan HTML y CSS de una forma simple, ya que se trata del primer acercamiento para poder empezar a aprender todo sobre el desarrollo front-end, por lo que esta sera una pagina muy sencilla, la cual solo busca cumplir el porposito de la tarea.

¿En qué consiste el desarrollo Fron-End?

A grandes rasgos, un desarrollador Font-End se encarga de crear y diseñar la parte visible de una aplicacion web, aquella con la cual el usuario tendrá contacto al momento de utilizarla, por lo que es importante que esta sea amigable y fácil de entender, asegurando la mejor experiencia posible.

Para el desarrollo Front-End se utilizan principalmente las siguientes tecnologias:

HTML, el cual es un lenguaje que permite definir y dar estructura a una pagina web, lo que podriamos considerar su "esqueleto".
Para saber más acerca de este lenguaje puede hacerlo presionando aquí

CSS, el cual es un lenguaje utilizado para aplicar estilos a una pagina web, las cuales se apican sobre el código HTML.
Para saber más acerca de este lenguaje puede hacerlo presionando aquí

JavaScript, el cual es un lenguaje utilizado para implementar funciones en una pagina web..
